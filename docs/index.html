OpenVINO · Intel NPU · Local Inference

intel_npu_llm

A Python CLI to download, load, and benchmark OpenVINO-optimized LLMs across CPU, GPU, and NPU, with persistent TTFT and TPS metrics.

View Repository Jump to Benchmarks

Project Overview

Interactive Model Ops

Select, download, delete, and load OpenVINO models from the CLI.

Device-Aware Metrics

Stats are tracked per model and per device with separate normal vs benchmark modes.

Repeatable Benchmarks

Run multi-prompt benchmarks across CPU/GPU/NPU and persist comparisons over time.

CLI Commands

Commands available from the terminal in the interactive CLI.

/help

Show command help.

/models

Select and load a model (downloads if missing).

/add_model

Add a new model interactively and save it to ov_models/models.json.

/delete

Delete model files from disk (model stays in the list).

/stats

Show metrics in separate tables for normal and benchmark modes.

/clear_stats

Clear all stored stats.

/clear_stats <n>

Clear all stats for model number n.

/clear_stats <n> <device>

Clear stats only for a model/device pair.

/current_model

Show current loaded model and runtime config.

/benchmark

Run benchmark for all models or only missing models (CPU/GPU/NPU).

/benchmark <n>

Run benchmark only for model number n across CPU/GPU/NPU.

/start_server

Start OpenAI-compatible backend server on port 1311.

/exit

Exit the CLI.

API Backend Compatibility

The CLI can run as a backend server compatible with OpenAI Chat Completions.

Endpoint: POST /v1/chat/completions

Default local URL: http://0.0.0.0:1311/v1/chat/completions

Command: /start_server

Behavior: same request/response shape as OpenAI chat completions (model, messages, max_tokens, temperature, top_p).
